What is Socket.io

What is socket.io:

+ Socket.IO is a library that enables real-time, bidirectional and event-based communication between the browser and the server.

+ It’s contains:

  • a Node.js server
  • a Javascript client library for the browser (which can be also run from Node.js)

The Node.js server I place in “socketIo” folder on .zip file. The client library I import in pod file of Xcode (in “ChatApp” folder).

Now we have all things we need to make a chat app.

*Note: About how the socket works please visits home page https://socket.io/docs/ or searching over internet.

Quick Start

To run an example please following below steps:

 

1. Extract ChatApp_SocketIo.zip file you will see 2 folders:

+ Folder “SocketIo” is the node.js server side

+ Folder “ChatApp” is Xcode file contains swift files and socket.io client library import via pod.

 

2. Run the server code by Node.js:

+ First you need to down load Node.js at https://nodejs.org/en/

+ Download Visual Studio Code https://code.visualstudio.com/ or which code editor you like -> Open Visual Studio Code 

-> In Visual Studio Code open folder “SocketIo” we just extracted from .zip file -> Open file “app.js” -> Change the IP address at line 6 with your -> Also in Visual Studio Code open terminal and write “nodemon app” -> Enter. If it’s showing line “[nodemon] starting `node app.js`” with no error then the server have been running correctly.

 

3. Now we have socket server. Next step we will run client side. 

+ First, we need to install pod files in project: Open terminal -> change directory to Xcode folder (ChatApp) -> write "pod install" -> enter and wait for it's finish running.

+ Open folder “ChatApp” and run file “SocketApp.xcworkspace” by Xcode -> In “LoginViewController.swift” file, replace the IP address at line 23 by your -> Run in 2 or 3 devices to testing -> Woalaa, We have the Chat App using socket.io.
